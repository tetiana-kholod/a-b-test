# a-b-test
A/B testing analysis project covering data preparation, statistical testing, visualization.

Tools: Python, SQL, Tableau

Goal: Evaluate how a proposed change impacts user behavior and key business metrics compared to the control version.

Dataset: 350K user sessions (anonymized).

Steps  
Data cleaning and preprocessing  
Exploratory analysis  
Statistical significance test  
Visualization of results
